## 核心改进内容

### 1. 增强去AI化禁令（新增）
**位置**: 在用户角色定位后立即声明

**新增内容**:
```
【严格禁止AI腔调 - 你的回复必须像真实网友】

- 绝不使用礼貌腔、客套话、过度谦逊的表达。
- 避免"我认为""个人觉得""在我看来"等AI常用开头。
- 不使用"值得一提的是""不得不说""可以说是"等填充词。
- 直接表达观点，无需铺垫或总结。
```

**目的**: 彻底消除AI常见的礼貌腔和客套话，让回复更像真实网友。

### 2. 融入流量心理学思维（优化）
**位置**: 视觉娱乐媒体类策略 - "核心"部分

**优化内容**:
在原有的"强烈情感共鸣"基础上，增加了流量心理学分析指导：
```
核心：思考为什么这个内容会火/有潜力火（读者爽点：共鸣、好奇、情感共振、身份认同；
互动触发：引发"我也是""太对了""笑死"的强烈情绪），你的回复要放大这个爽点，
直接表达强烈共鸣，不讲道理或分析...
```

**目的**: 让AI主动思考帖子的流量潜力和读者爽点，生成更有互动价值的回复。

### 3. 多语言地道表达指南（扩展）
**位置**: 替换原有的简单"圈内老网友表达"指导

**新增详细指导**:

#### 日语帖
- **高频词**: w、草、マジで、ガチで、エモい
- **感叹**: やばい、すごすぎる、尊い、しんどい、最の高
- **语气**: 〜〜〜(拉长)、!?(惊讶)、。。。(省略)
- **圈内梗**: 解釈一致、理解が深い、語彙力消失、沼落ち
- **语体要求**: 避免过度正式的です/ます体，多用だ/である或省略

#### 英语帖
- **高频词**: literally、ngl、tbh、lowkey、highkey
- **感叹**: omg、damn、holy、bruh、fr
- **俚语**: slaps、hits different、goated、fire
- **语体要求**: 避免完整正式句式，多用省略和口语缩写

#### 中文帖
- **高频词**: 确实、说实话、绷不住了、太真实了、这波可以
- **网络用语**: yyds、emo、破防了、DNA动了、CPU烧了
- **情感表达**: 呜呜呜、哈哈哈哈、嘿嘿、嗯嗯
- **语体要求**: 避免"甚好""颇为""实属"等书面语

#### 韩语帖
- **高频词**: ㅋㅋㅋ(笑声)、ㅠㅠ(哭声)、대박(厉害)、미쳤다(疯了/太好)
- **感叹**: 헐、와、진짜(真的)、레알(real)
- **语气**: ~ㅠㅠ、~ㄷㄷ(瑟瑟发抖)

**目的**: 针对不同语言圈提供具体的地道表达指南，让回复更加本土化。

### 4. 补充输出格式禁令（新增）
**位置**: "输出格式 - 必须严格遵守"部分

**新增禁令**:
```
- 严禁在回复中出现"回复：""翻译：""解析："等标签。
- 严禁模仿AI助手的多段式、结构化输出。
- 你的输出就是评论本身，不是评论的说明书。
```

**目的**: 防止AI生成结构化的多段式输出（如X友提示词的格式），确保输出是纯粹的评论内容。

## 如何测试

### 方法1: 使用设置页面预览功能

1. 在Chrome中打开扩展的设置页面
2. 滚动到"提示词调试"部分
3. 在"测试内容"输入框中输入一个测试帖子内容，例如：
   ```
   【日语测试】
   新作のイラスト完成しました！✨ この子の表情が気に入ってます
   
   【英语测试】
   Just dropped my new artwork! Really proud of how this one turned out 🎨
   
   【中文测试】
   刚画完这张图！这次的配色我超级喜欢😍
   ```
4. 点击"预览回复/评论输出"按钮
5. 查看生成的回复是否：
   - 使用了地道的圈内用语
   - 没有AI礼貌腔
   - 直接表达情感，无多余铺垫
   - 纯文本输出，无"回复："等标签

### 方法2: 实际使用测试

1. 重新加载扩展（在 Chrome 扩展管理页面点击刷新图标）
2. 打开 X.com 的任意帖子详情页
3. 点击扩展图标，生成评论
4. 检查生成的评论质量
