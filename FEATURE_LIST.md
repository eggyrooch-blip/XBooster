# XBooster - Chrome Web Store 功能清单

## 扩展基本信息

- **名称**：XBooster
- **版本**：1.0.0
- **描述**：智能生成 X（Twitter）评论，支持批量处理、情绪选择、翻译等功能
- **平台**：Chrome / Edge / 其他基于 Chromium 的浏览器
- **支持网站**：x.com, pro.x.com

---

## 核心功能清单

### 1. 智能评论生成 ⭐⭐⭐⭐⭐

#### 1.1 单条回复生成
- ✅ 自动读取当前帖子内容
- ✅ 基于 AI 生成个性化评论
- ✅ 支持多种情绪风格
- ✅ 一键复制到剪贴板
- ✅ 一键填入回复框
- ✅ 支持自定义提示词模板

#### 1.2 批量回复生成
- ✅ 右下角浮窗界面
- ✅ 自动扫描页面上的帖子
- ✅ 队列并发处理（默认并发数：2）
- ✅ 自动续扫新加载的帖子
- ✅ 结果卡片展示（待处理/已接受/失败）
- ✅ 每条评论可单独操作（填入/复制）
- ✅ 支持重试失败的评论
- ✅ 实时状态更新

#### 2.1 情绪风格
- ✅ 友好（😊）：温暖、支持、积极
- ✅ 专业（💼）：正式、客观、专业
- ✅ 提问（❓）：好奇、探索、引导
- ✅ 幽默（😄）：轻松、有趣、幽默
- ✅ 深度（🤔）：深入、分析、洞察
- ✅ 简洁（⚡）：简短、有力、精炼
- ✅ 批判（🧐）：批判性、质疑、建设性

#### 2.2 功能特性
- ✅ 可视化情绪选择界面

### 3. 智能翻译功能 ⭐⭐⭐⭐

#### 3.1 翻译模式
- ✅ 自动翻译（可配置开关）
- ✅ 手动翻译（按需触发）
- ✅ 帖子内容翻译
- ✅ 评论内容翻译

## 权限说明

### 必需权限
- `activeTab`：访问当前活动标签页
- `storage`：保存配置和统计数据
- `scripting`：注入内容脚本

### 主机权限
- `https://x.com/*`：访问 X 平台
- `https://pro.x.com/*`：访问 X Pro 平台
- `https://api.openai.com/*`：访问 OpenAI API
- `https://api.siliconflow.cn/*`：访问 SiliconFlow API
- `https://*/*`：访问自定义 API 接口

## 使用场景

1. **社交媒体管理**：批量回复粉丝评论，提高互动效率
2. **内容创作**：快速生成高质量评论，节省时间
3. **多语言交流**：自动翻译帖子，打破语言障碍
4. **情绪表达**：根据不同场景选择合适的评论语气
5. **数据分析**：通过统计看板了解评论处理情况
