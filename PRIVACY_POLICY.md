# XBooster 隐私政策

**最后更新日期：2025年1月14日**

## 概述

XBooster（"本扩展"、"我们"）致力于保护您的隐私。本隐私政策说明了我们如何收集、使用和保护您的信息。

## 数据收集与存储

### 本地存储

本扩展使用 Chrome 浏览器的本地存储功能（Chrome Storage API）来保存以下数据：

- **配置信息**：包括 API Key、模型选择、语言设置、提示词模板等配置项
- **统计数据**：包括每日处理的评论数量、成功/失败统计等
- **用户偏好**：包括当前选择的情绪风格、翻译设置等

**重要说明**：
- 所有数据仅存储在您的本地浏览器中
- 我们不会将任何数据上传到我们的服务器
- 数据不会与第三方共享

### API Key 存储

- API Key 仅存储在您的本地浏览器中
- API Key 仅用于调用您配置的 AI 接口（OpenAI 或自定义接口）
- 我们不会收集、存储或访问您的 API Key

## 数据传输

### AI 接口调用

当您使用本扩展生成评论或翻译时：

- 帖子内容和您的提示词会发送到您配置的 AI 接口（OpenAI 或自定义接口）
- 这些数据仅用于生成评论和翻译，不会用于其他目的
- 数据传输使用 HTTPS 加密协议

### 翻译服务

本扩展可能使用以下翻译服务：

- Google Translate API（免费版）
- 您配置的自定义翻译接口

翻译内容仅发送到相应的翻译服务，不会存储在我们的服务器上。

## 第三方服务

本扩展可能使用以下第三方服务：

- **OpenAI API**：用于生成评论内容
- **Google Translate**：用于翻译功能
- **自定义 API 接口**：由您配置的第三方服务

我们不对这些第三方服务的隐私政策负责。建议您查看这些服务的隐私政策。

## 数据安全

- 所有配置数据存储在本地，不会上传到服务器
- API Key 等敏感信息仅存储在您的本地浏览器中
- 我们不会收集您的个人信息、浏览历史或其他敏感数据

## 权限说明

本扩展请求以下权限：

- **activeTab**：用于访问当前活动标签页，读取帖子内容
- **storage**：用于保存您的配置和统计数据（仅本地存储）
- **scripting**：用于在 X（Twitter）页面上注入必要的功能脚本
- **host_permissions**：用于访问 x.com、pro.x.com 以及您配置的 API 接口

## 数据删除

您可以随时通过以下方式删除本扩展存储的数据：

1. 卸载扩展：卸载扩展会自动清除所有本地存储的数据
2. Chrome 设置：通过 `chrome://extensions/` 页面清除扩展数据

## 儿童隐私

本扩展不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。

## 隐私政策更新

我们可能会不定期更新本隐私政策。更新后的隐私政策将在本页面发布，并更新"最后更新日期"。

## 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

- GitHub Issues：通过项目仓库的 Issues 页面提交问题
- 项目仓库：访问项目的 GitHub 仓库页面

## 同意

使用本扩展即表示您同意本隐私政策的条款。如果您不同意本隐私政策，请不要使用本扩展。

---

**注意**：本隐私政策仅适用于 XBooster Chrome 扩展。我们不对您访问的第三方网站（如 X/Twitter）的隐私政策负责。
